# Revamp Feedback — expzeno agent review

**Date received:** 2026-05-07 (via expzeno agent review of siteez.labzeno.com)

## What's working
- Dark gradient theme — modern, on-brand for fintech
- Hero headline: "Top up $50. Spend $55. Keep the difference." — lands value prop cleanly
- Floating app UI mockup adds credibility

## What needs work (priority order)

1. **Hero phone mockup is too small** — "$284.50" wallet card is the actual product; should be larger and more legible
2. **Sections run together** — uniform dark background top to bottom, no visual anchors or contrast between sections
3. **"Simple steps, powerful results"** — generic fintech boilerplate; more specific copy preferred ("Live in one afternoon", "From install to first payment in under an hour")
4. **Tier section looks sparse** — thresholds TBD, section looks incomplete; consider replacing with a simpler "the more they spend, the more they earn" graphic instead of named tiers
5. **Testimonials** — if placeholders, cut them entirely; fake-looking social proof is worse than none
6. **FAQ** — text-heavy answers on dark background need extra line-height and contrast
7. **No social proof numbers** — "X merchants, Y customers, Z transactions" in the hero would add weight

## Revamp priorities (from review)
- Hero mockup: bigger, more interactive-looking app UI (animated wallet balance)
- Section contrast: alternating section treatments to break monotone dark
- Cut boilerplate (fake testimonials if not real)

## Status
Lion had not given scope direction (design refresh vs full copy overhaul) when this feedback arrived. Stand by for instructions.
