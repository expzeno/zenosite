SECURITY & TRUST

Built on an unbreakable foundation

Every balance, every transaction, every loyalty point is protected by enterprise-grade financial infrastructure — not an afterthought.

⛓️
SHA-256 Hash Chain

Every transaction is chained to the previous one with a cryptographic hash — making any tampering instantly detectable.

📒
Double-Entry Bookkeeping

Every wallet movement has a matching ledger entry. Automated reconciliation runs continuously to verify every cent balances.

🔑
JWT RS256 Auth

2048-bit RSA key authentication with short-lived tokens, refresh rotation, and version-based revocation for stolen session protection.

🔒
Pessimistic Locking

Concurrent wallet operations use database-level row locking to prevent race conditions — your balance is always exactly right.

🛡️
AML Review

Large withdrawals trigger an automatic AML hold for admin review before funds are released — protecting merchants and customers.

👆
Biometric Authentication

Face ID, Touch ID, and device PIN protect your wallet session. All payments and session restores require biometric verification.

🔐
Wallet Lock Mode

Freeze your wallet instantly to block all new payments while keeping existing preauth reservations and topups fully active.

🏦
PDPA Compliant

Built to Singapore's Personal Data Protection Act 2012. No payment card numbers or bank data stored — only what's needed to run the platform.
